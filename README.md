# CONNEX Archive | Aether Glass Edition

A high-end architectural lighting portfolio built with **React**, **TypeScript**, **Framer Motion**, and **Vite**.

## 🌌 Overview
This project is a high-performance, cinematic digital experience designed for **Connex**, an architectural lighting studio based in Berlin. It features a sophisticated "Aether Glass" aesthetic, focusing on optical depth, glassmorphism, and technical precision.

## 🛠 Key Features
- **Aether Glass UI**: A deep indigo/ink palette with iridescent highlights and spectral ghosting.
- **Glass Vault Grid**: Interactive 3D product modules with micro-tilt physics and subsurface light tracking.
- **Refractive Hero**: Post-processed typography with internal glows and cinematic lens flares.
- **Museum Gallery Narrative**: A smooth, desynchronized stroll through the product archive and engineering philosophy.
- **Liquid Motion**: Spring-physics based scroll transitions for an organic, premium feel.

## 🚀 Tech Stack
- **Framework**: React 19 + TypeScript
- **Styling**: Tailwind CSS 4.0
- **Animation**: Framer Motion
- **Performance**: Vite HMR
- **Scrolling**: Lenis (Smooth Scroll)

## 📦 Navigation
- **Hero**: Brand launchpad and optical narrative.
- **Philosophy**: Engineering values and studio vision.
- **Intelligence**: Technical specifications and metrics.
- **Archive**: The interactive collection of luminaires.
- **Protocol**: High-end contact and enquiry portal.

## 🚦 Getting Started
1. `npm install`
2. `npm run dev`
